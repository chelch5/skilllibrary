# Purpose
Writes internal announcements, updates, and stakeholder communications clearly and consistently.

# When to use this skill
Use when:
- Writing internal announcements or team updates
- Drafting stakeholder communication for project milestones
- Creating internal release or change communications

Do NOT use when:
- External-facing marketing or PR communications
- Technical documentation for end users

# Operating procedure
1. Identify audience (team, org, leadership)
2. Define message purpose and key takeaways
3. Write with appropriate tone (formal, friendly, urgent)
4. Include clear call-to-action or next steps
5. Review for clarity and completeness

# Output defaults
Internal communication draft with subject, body, key points, and clear CTA

# Failure handling
If audience scope is unclear, target the most restrictive audience. Escalate if needed.
