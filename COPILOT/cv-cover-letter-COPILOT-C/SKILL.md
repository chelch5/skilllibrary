# Purpose
Writes or refines CVs and cover letters tailored to specific roles and companies.

# When to use this skill
Use when:
- Writing or improving a CV or resume for job applications
- Drafting a cover letter tailored to a specific role
- Adapting application materials for different industries

Do NOT use when:
- Technical portfolio content (separate from job applications)
- Internal performance reviews or bios

# Operating procedure
1. Understand target role and company values
2. Tailor CV to highlight relevant experience
3. Write cover letter with specific role connection
4. Quantify achievements with metrics where possible
5. Review for ATS keyword coverage

# Output defaults
CV document and cover letter tailored to specific role, with quantified achievements

# Failure handling
If experience gaps exist, emphasize transferable skills. Never fabricate credentials.
